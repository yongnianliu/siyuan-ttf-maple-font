# 思源笔记 Maple Mono 字体插件

为[思源笔记](https://github.com/siyuan-note/siyuan)提供 [Maple Mono](https://github.com/subframe7536/maple-font) 等宽字体支持的插件。

## 特性

- **完整字重支持**：包含从 Thin (100) 到 ExtraBold (800) 的全部 8 种字重
- **斜体支持**：所有字重均包含斜体变体
- **代码优化**：专为代码块和编程场景优化
- **连字支持**：支持美观的编程连字
- **多语言支持**：适配英文、简体中文、繁体中文和日文界面

## 字重对照表

| 字重 | 数值 | 文件 |
|------|------|------|
| Thin（极细） | 100 | MapleMono-Thin |
| ExtraLight（特细） | 200 | MapleMono-ExtraLight |
| Light（细体） | 300 | MapleMono-Light |
| Regular（常规） | 400 | MapleMono-Regular |
| Medium（中等） | 500 | MapleMono-Medium |
| SemiBold（半粗） | 600 | MapleMono-SemiBold |
| Bold（粗体） | 700 | MapleMono-Bold |
| ExtraBold（特粗） | 800 | MapleMono-ExtraBold |

## 注意事项

本插件可能与集市中的其他字体或 Emoji 插件不兼容，使用本插件时请禁用其他字体或 Emoji 插件。

## 安装方法

1. 从思源集市下载插件
2. 在 设置 → 集市 → 插件 中启用插件
