# SiYuan Maple Mono Font Plugin

A font plugin for [SiYuan Note](https://github.com/siyuan-note/siyuan) that uses [Maple Mono](https://github.com/subframe7536/maple-font) as the interface and code font.

## Features

- **Complete Weight Support**: Includes all 8 font weights from Thin (100) to ExtraBold (800)
- **Italic Support**: All weights include italic variants
- **Optimized for Code**: Perfect for code blocks and programming
- **Ligature Support**: Beautiful programming ligatures enabled
- **Multi-language Support**: Works with English, Chinese (Simplified/Traditional), and Japanese interfaces

## Font Weights

| Weight | Value | File |
|--------|-------|------|
| Thin | 100 | MapleMono-Thin |
| ExtraLight | 200 | MapleMono-ExtraLight |
| Light | 300 | MapleMono-Light |
| Regular | 400 | MapleMono-Regular |
| Medium | 500 | MapleMono-Medium |
| SemiBold | 600 | MapleMono-SemiBold |
| Bold | 700 | MapleMono-Bold |
| ExtraBold | 800 | MapleMono-ExtraBold |

## Notice

This plugin may not be compatible with other font or Emoji plugins in the bazaar. Please disable other font or Emoji plugins when using this plugin.

## Installation

1. Download the plugin from SiYuan Bazaar
2. Enable the plugin in Settings → Bazaar → Plugins
