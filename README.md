# SiYuan Maple Mono NF CN Font Plugin

A font plugin for [SiYuan Note](https://github.com/siyuan-note/siyuan) that uses [Maple Mono NF CN](https://github.com/subframe7536/maple-font) (with Nerd Font icons and Chinese support) as the interface and code font.

## Features

- **Full Chinese Support**: Complete Chinese character set included
- **Nerd Font Icons**: Built-in Nerd Font icons, perfect for developers
- **Complete Weight Support**: Includes all 8 font weights from Thin (100) to ExtraBold (800)
- **Italic Support**: All weights include italic variants
- **Optimized for Code**: Perfect for code blocks and programming
- **Ligature Support**: Beautiful programming ligatures enabled
- **Multi-language Support**: Works with English, Chinese (Simplified/Traditional), and Japanese interfaces

## Font Weights

| Weight | Value | File |
|--------|-------|------|
| Thin | 100 | MapleMono-NF-CN-Thin |
| ExtraLight | 200 | MapleMono-NF-CN-ExtraLight |
| Light | 300 | MapleMono-NF-CN-Light |
| Regular | 400 | MapleMono-NF-CN-Regular |
| Medium | 500 | MapleMono-NF-CN-Medium |
| SemiBold | 600 | MapleMono-NF-CN-SemiBold |
| Bold | 700 | MapleMono-NF-CN-Bold |
| ExtraBold | 800 | MapleMono-NF-CN-ExtraBold |

## Notice

- This plugin may not be compatible with other font or Emoji plugins in the bazaar. Please disable other font or Emoji plugins when using this plugin.
- Due to the complete Chinese character set, font files are larger (~6MB each), first load may take some time.

## Installation

1. Download the plugin from SiYuan Bazaar
2. Enable the plugin in Settings → Bazaar → Plugins
